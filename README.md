## **Installation 🧰**

1. Clone the repository (you need to have [Git](https://git-scm.com) installed).

```shell
    git clone https://github.com/Franklin361/multi-lang-app.git
```

2.  Install dependencies of the project.

```shell
    npm install
```

3. Run the project.

```shell
    npm run dev
```
